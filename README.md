# environment-setup-tools

Scripts for installing development tools

## Usage

### Install Git

<https://git-scm.com/install/mac>

```bash
brew install git
```

<https://git-scm.com/install/linux>

```bash
apt-get install git
```

### Clone this repository

```bash
git clone https://github.com/hashin2425/environment-setup-tools.git
```
